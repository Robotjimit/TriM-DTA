## 快速使用

### 环境要求

- Python 版本: >= 3.7
- 依赖库: 请参考 `requirements.txt` 或根据项目需要安装相关依赖。

### 安装依赖

```bash
pip install -r requirements.txt
```

### 数据准备

将数据文件放置在项目根目录下，确保文件名为 `data.csv`。

### 运行项目

运行以下命令以启动项目：

```bash
python main.py
```
